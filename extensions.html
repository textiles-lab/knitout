Extensions for the "Knitout" (.k) File Format

This file contains a summary of the known extensions for the knitting output file format ("Knitout" / .k). If you have developed an extension and would like it to be included, please use the Issues page, or send e-mail to the knitout-feedback@cs.cmu.edu alias. 

  • x-stitch-number 5
    Stitch Number CMU Textiles Lab / SWG*N2 Backend. The 'x-stitch-number' command sets the stitch number (an abstract index into a machine table of stitch and leading values). 
  • x-speed-number 10
    Speed Number CMU Textiles Lab / SWG*N2 Backend. The 'x-speed-number' command sets the speed number (an abstract index into a machine table of speed values). Sets speed for both knitting and transfer passes. Valid speeds are 0 (default) or 1-15.
  • x-sub-roller-number 3
    Sub-Roller Number CMU Textiles Lab / SWG*N2 Backend. The 'x-sub-roller-number' command sets the sub roller table number (an index into a machine table of sub roller settings). Sets speed for only knitting passes. Valid speeds are 0 (default) or 1-31.
  • x-presser-mode auto
    Fabric Presser Mode CMU Textiles Lab / SWG*N2 Backend. The 'x-presser-mode' command sets the fabric presser mode to 'off' (no fabric presser), 'auto' (fabric presser on passes with only front or only back stitches), or 'on' (break passes so that fabric presser can always be used). The default mode is 'off'.
